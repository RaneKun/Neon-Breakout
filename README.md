## 🌟 Features

### 🎯 Core Gameplay
- **10 Hand-Tuned Stages** - Each with 5 unique brick patterns
- **Endless Mode** - Unlocks after game clear and every stage here is random and never ending!
- **Escalating Difficulty** - More HP, unbreakable bricks, and faster ball speed as you progress
- **Time Pressure** - Each stage has a time limit that triggers a **difficulty spike** when it expires!
- **Persistent Progress** - Your best stage reached and high scores are saved automatically

### ⚡ Power-Ups (Rare & Random)
| Power-Up | Effect | Rarity |
|----------|--------|--------|
| 🔵 **Widen** | Paddle grows for 9 seconds | Medium |
| 🟣 **Multiball** | 3 balls launch simultaneously | High |
| 🔷 **Slow** | Ball speed reduced for 6 seconds | Medium |
| ⚪ **Extra Life** | +1 life (max 6) | Medium |
| 🔴 **Shrink** | Paddle shrinks for 7 seconds (bad!) | Medium |
| 🟠 **Fast** | Ball speed boosted for 5 seconds (risk!) | Medium |

### 🎨 Visual & Audio
- **Neon Glow Effects** - Everything pulses with retro-futuristic neon vibes
- **Particle System** - Explosive visual feedback on every hit
- **Procedural Music** - ~2 minutes of evolving synthwave/chiptune OST (no external audio files!)
- **Synthesized SFX** - All sound effects generated in code
- **Resizable Window** - Scales beautifully to any monitor size

### 🎮 Controls
| Action | Keys |
|--------|------|
| Move Paddle | ← → or A/D or **Mouse** |
| Launch Ball | SPACE or **Left Click** |
| Pause | P, ESC, or **click pause icon** (top-left) |
| Toggle Cursor | CTRL or ALT |
| Restart | R (on Game Over / Victory) |
| Quick Mute | M |
| Quit | ESC (from main menu) |

## 🚀 Quick Start

### 1. Install Dependencies

```bash
pip install pygame numpy
```

### 2. Run the Game

```bash
python neon_breakout.py
```

### 3. Play!

- Navigate the main menu
- Choose **START** to begin from Stage 1
- Or **CONTINUE** to pick up from your best progress
- Unlock **STAGE SELECT** and **ENDLESS MODE** by beating all 10 stages!
